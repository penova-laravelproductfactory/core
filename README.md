# Penova Core

A powerful core for Laravel projects – modular, production-ready starter base.

## What is this?

Penova Core is a Laravel + Vue + Inertia + Tailwind boilerplate designed to be the foundation for multiple products:
- Authentication, users, roles & permissions
- Admin layout and reusable UI components
- Basic settings, notifications, logs and API support

## Who is it for?

- Laravel developers who are tired of rebuilding the same boilerplate  
- Agencies and teams that start several Laravel projects each year  
- Freelancers who want to focus on business logic instead of setup
